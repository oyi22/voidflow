# Voidflow

Voidflow adalah platform modern untuk membangun dan meluncurkan produk digital yang cepat, aman, dan scalable — mulai dari website sederhana hingga aplikasi web kompleks dengan arsitektur yang solid.

---

## Features

* Dibangun dengan Next.js (App Router)
* Menggunakan TypeScript
* UI modern dan responsif
* Performa optimal dan SEO-friendly
* Arsitektur scalable

---

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Vercel

---

## Getting Started

### 1. Clone repository

```
git clone https://github.com/oyi22/voidflow.git
cd voidflow
```

### 2. Install dependencies

```
npm install
# atau
yarn install
# atau
pnpm install
# atau
bun install
```

### 3. Jalankan development server

```
npm run dev
```

### 4. Buka di browser

```
http://localhost:3000
```

---

## Project Structure

```
/app        → Routing dan halaman (App Router)
/components → Komponen UI reusable
/lib        → Utility function
/public     → Asset statis
/styles     → Styling global
```

---

## Development

Untuk mulai mengedit halaman utama:

```
app/page.tsx
```

Perubahan akan langsung terlihat di browser (hot reload).

---

## Deployment

Direkomendasikan menggunakan Vercel untuk deployment:

```
npm run build
npm start
```
